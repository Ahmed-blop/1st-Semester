Form Input Types Explanation

Text Input: Used for single-line text fields.

Password Input: Characters are masked (shown as dots or asterisks).

Radio Buttons: Used when the user must select exactly one option from a list.

Checkboxes: Used when the user can select zero or more options.

Date Input: Allows the user to select a date.

Submit Button: A button for submitting the form data to a form-handler.
