Introduction to HTML

HTML stands for HyperText Markup Language. It is the standard markup language for creating Web pages.

HTML describes the structure of a Web page and consists of a series of elements. Elements tell the browser how to display the content, for example, "this is a heading", "this is a paragraph", or "this is a link".
